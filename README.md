### Cloning the repository

```shell
git clone https://github.com/tan8325/habitu.git
```

### Install packages

```shell
npm i
```

### Setup .env file

```js
# Deployment used by `npx convex dev`
CONVEX_DEPLOYMENT=
NEXT_PUBLIC_CONVEX_URL=

NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=
CLERK_SECRET_KEY=

```

### Setup Convex

```shell
npx convex dev

```
### Start the app

```shell
npm run dev
```
